# 03. 목록
<hr>

### 순서가 있는 목록
- 숫자와 점을 사용
```
1. 첫번쨰
2. 두번쨰
3. 세번쨰
```

1. 첫번째
2. 두번째
3. 세번째


- 순서에 맞지 않게 입력해도 내림차순으로 정의
```
1. 첫번째
3. 세번째
2. 두번째
```


1. 첫번째
3. 세번째
2. 두번째

<hr>

### 순서가 없는 목록
- 글머리기호 : ```*, +, -``` 가능
```
* 빨강
  * 녹색
    * 파랑

+ 빨강
  + 녹색
    + 파랑

- 빨강
  - 녹색
    - 파랑
```
* 빨강
    * 녹색
        * 파랑

+ 빨강
    + 녹색
        + 파랑

- 빨강
    - 녹색
        - 파랑



- 혼합하여 사용가능
```
* 1단계
  - 2단계
    + 3단계
      * 4단계
```

* 1단계
    - 2단계
        + 3단계
            * 4단계
